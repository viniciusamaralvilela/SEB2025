> 📁 Repositório da disciplina **Sistemas Embarcados – 2025**  
> 💻 Projetos práticos focados em programação para microcontroladores STM32, uso de displays, controle de GPIO e lógica de programação em linguagem C.

---

## 📌 O que é este projeto?

Este repositório reúne os projetos e exercícios práticos realizados na disciplina SEB em 2025.  
Aqui você encontrará códigos em C para STM32, exemplos de controle de displays (ST7789), jogos simples como memória e lógica com botões e LEDs.

Ideal para quem quer aprender programação embarcada e desenvolvimento com STM32!

---

## 🛠️ Tecnologias Utilizadas

- 🔹 **STM32F103C8** — microcontrolador  
- 🔹 **Display ST7789/ST7735** — interface gráfica  
- 🔹 **Linguagem C** — para programação embarcada  
- 🔹 **STM32CubeIDE** — ambientes para desenvolvimento e upload do código

---

## 💻 Como instalar e usar?

### ⚙️ Pré-requisitos

- STM32CubeIDE ou outro ambiente de desenvolvimento para STM32  
- Git instalado (opcional)  
- Hardware STM32F103C8 e display ST7789 ou display ST7735 (para testes físicos)

### 📥 Instalando o projeto

```bash
git clone https://github.com/viniciusamaralvilela/SEB2025.git
cd DPL_2025
```

Abra a pasta do projeto desejado na sua IDE (ex: STM32CubeIDE).

## ▶️ Como usar?

1. Escolha o projeto que deseja abrir (exemplo: `GIF`).  
2. Abra no STM32CubeIDE ou ambiente compatível.  
3. Compile e envie o programa para o microcontrolador.  
4. Faça os testes com os componentes físicos conectados (display, botões, LEDs).
